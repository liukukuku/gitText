
		git 命令
git init 初始化git仓库

git push -u origin '项目名称‘ 第一次将项目放入远程仓库

git config --global user.name "youname" 设置git用户名

git config --global user.email "邮箱地址" 设置git邮箱

git config --list 查看git配置

git remote add origin https://github.com/liukukuku/Myremote.git 将本地仓库链接到远程仓库 或者叫载入远程仓库

git add .   是上传当前所有文件到仓库 .代表当前所有文件

git commit -m '注释信息'

git push 上传远程代码并合并

git status 查看仓库当前的状态 如果有红色的 就是没有上传到仓库 需要进行上传到仓库 否则不能commit

git checkout 要跳转的分支名称 跳转分支

git checkout -b 要创建并跳转的分支名称

git branch -D 分支名 删除当前分支 不能删除当前所在的分支

git branch 查看当前所有分支
